Given: POSIX and virtualenv

Step 1: Make a sandbox:

$ virtualenv foo
$ cd foo
$ . bin/activate

Step 3: Install aspen from PyPI:

(foo)$ pip install aspen

Step 2: Create a website root:

$ mkdir www
$ cd www

Step 4: Create a web page, and start aspen inside it:

(foo)$ echo Greetings, program! > index.html
(foo)$ aspen
Greetings, program! Welcome to port 8080.

Step 5: Check localhost for your new page!
